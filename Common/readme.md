# Coding Interview Patterns and Problems

This README outlines a curated list of common coding problems, categorized by difficulty levels. Each level focuses on specific data structures and algorithms, helping you build a strong foundation for technical interviews.

## 🟢 Fresher Level (Foundations)

**Focus:** Arrays, Strings, Basic Hashing

### 🔹 Arrays & Strings
- Two Sum
- Best Time to Buy and Sell Stock
- Maximum Subarray (Kadane’s Algorithm)
- Move Zeroes
- Contains Duplicate
- Product of Array Except Self

### 🔹 Strings
- Valid Anagram
- Longest Common Prefix
- Valid Palindrome
- First Unique Character

### 🔹 Hashing
- Majority Element
- Intersection of Two Arrays

## 🟡 Intermediate Level (Core Problem Solving)

**Focus:** Sliding Window, Recursion, Linked List, Stack, Queue

### 🔹 Sliding Window / Two Pointers
- Longest Substring Without Repeating Characters
- Minimum Window Substring
- Container With Most Water

### 🔹 Linked List
- Reverse Linked List
- Detect Cycle in Linked List (Floyd’s Cycle)
- Merge Two Sorted Lists

### 🔹 Stack / Queue
- Valid Parentheses
- Min Stack
- Daily Temperatures
- Implement Queue using Stacks

### 🔹 Recursion & Backtracking
- Subsets
- Permutations
- Combination Sum

## 🔵 Advanced Level (High Interview Frequency)

**Focus:** Trees, Graphs, Dynamic Programming

### 🔹 Trees & BST
- Binary Tree Level Order Traversal
- Maximum Depth of Binary Tree
- Lowest Common Ancestor (LCA)
- Validate Binary Search Tree

### 🔹 Graphs
- Number of Islands (DFS/BFS)
- Clone Graph
- Course Schedule (Topological Sort)
- Detect Cycle in Graph

### 🔹 Dynamic Programming
- Climbing Stairs
- House Robber
- Longest Increasing Subsequence
- Coin Change
- Longest Common Subsequence

## 🔴 Senior Level (Optimization + System Thinking)

**Focus:** Advanced DP, Graph optimization, Design patterns

### 🔹 Advanced Problems
- Word Ladder
- Trapping Rain Water
- Median of Two Sorted Arrays
- Merge K Sorted Lists

### 🔹 DP Hard
- Edit Distance
- DP on Trees
- Palindromic Substrings / Partitioning

### 🔹 Graph Advanced
- Dijkstra’s Algorithm
- Bellman-Ford
- Union Find (Disjoint Set)

## 🟣 Lead / Architect Level (Rare but High Value)

**Focus:** Problem modeling + scalability thinking

### 🔹 Patterns + Real-world Mapping
- Design LRU Cache
- Design Autocomplete System
- Distributed Top-K Elements
- Stream Processing (Sliding Window on streams)
- Consistent Hashing

## ⚡ Must-Know Patterns (VERY IMPORTANT)

These patterns cover ~80% of interviews:

- Sliding Window
- Two Pointers
- Fast & Slow Pointer
- Binary Search on Answer
- Backtracking
- Dynamic Programming (1D / 2D)
- Graph Traversal (DFS/BFS)
- Greedy
- Heap / Priority Queue

Happy coding! Practice these problems on platforms like LeetCode to master the concepts.
